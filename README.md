# 🎬 Projeto Alura+ (Clone Responsivo)

Este projeto é uma página web inspirada na plataforma Alura+, desenvolvida com foco em prática de HTML, CSS e responsividade.

## 📌 Objetivo

Criar uma interface moderna e responsiva, simulando uma landing page de streaming educacional, com adaptação para diferentes tamanhos de tela (desktop e mobile).

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* Flexbox
* Grid Layout

---

## 📱 Responsividade

O projeto foi adaptado para diferentes dispositivos, incluindo:

* 📱 Celulares
* 💻 Notebooks
* 🖥️ Monitores maiores

Foram utilizadas **media queries** para garantir uma boa experiência em telas menores.

---

## 🎯 Funcionalidades

* Layout moderno inspirado em plataformas reais
* Seções organizadas (principal, secundárias, dispositivos, rodapé)
* Botões estilizados com hover
* Imagens responsivas
* Ajustes de tipografia para diferentes telas
